📊 Kia Motors Sales Analysis Project 
📌 Project Overview
This project analyses the sale data of Kia Motors usign SQL to extract meaningful insights of customer behaviour, product trends, delership performance.
🎯 Objectives
* Analyze sales peroformance across dealerships
* Identify top-selling and least-selling car models
* Uderstand customer purchasing behaviour
* Evaluate revenue trends
* Compare fule type demand

🗂️ Database Structure
This project uses the following tables
- Sales -> Transaction-level data
- Model -> Car Details such model, fuel type, and price
- Customer -> Customer information
- Dealership -> Dealership details

🔍 Key SQL Analysis Performed
+ Total cars sold by each delership
+ Monthly sales trend
+ Most sold car model
+ Delearship with highest sales

💰 Revenue Analysis
- Revenue generated by each car model
- Revenue by delearship
- Revenu by fule type
- Most profitable car

🚗 Product Insights
* Top three expensive and cheapest cars
* Latest launched models
* Fule type distribution

👥 Customer Insights
- Customers with multiple purchases
- Customers who never purchases
- Customers who bought electric cars
